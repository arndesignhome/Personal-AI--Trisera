Trisera AI,

Trisera AI is an advanced, multi-faceted artificial intelligence platform engineered to eliminate workflow friction and automate complex tasks with precision and speed. 

This project is proudly built by the Algorithm Warriors team, led by Developer Abdur Rahman alongside core team member Jhuma Akter. Together, their collaborative vision focuses on developing high-performance, user-centric tech solutions to tackle modern digital challenges.


To run Trisera AI on your PC, download or clone the repository files to your local machine. Open your terminal or command prompt in the project directory, ensure Python is installed, and run pip install -r requirements.txt to install the required dependencies. Once the setup is complete, simply launch the platform by executing the command python main.py.
